# route-optimizer
Personal Project of a Fullstack Web Application that Finds the Most Efficient Route between Multiple Addresses and Suggests Popular Tourist Sites to Assist Users with Creating Tourist Routes.

## Technologies Used

- Java 
- JavaScript
- HTML/CSS
- React
- Spring Boot 
- PostgreSQL
